
# React Native Todo App

간단한 투두 앱 프로젝트입니다. 

## 프로젝트 소개
- **목적**: React Native 학습 및 포트폴리오용
- **주요 기능**: 할 일 추가, 삭제, 수정 (추후 업데이트 가능)

## 사용 기술
- **React Native** (Expo)
- **JavaScript (ES6+)**  
- ~~기타 라이브러리~~ (사용하게 된다면 기입 할 예정띠리따라뽀)

# 실행 방법
1. 이 저장소를 클론 혹은 다운로드합니다.
2. 프로젝트 폴더에서 `npm install` 혹은 `yarn install`을 실행합니다.
3. `npm start` 또는 `yarn start` 후, Expo Go 앱 혹은 시뮬레이터를 통해 앱을 확인할 수 있습니다.

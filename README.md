
# React Native Todo App

간단한 투두 앱 프로젝트입니다. 

## 프로젝트 소개
- **목적**: React Native 학습 및 포트폴리오용
- **주요 기능**: 할 일 추가, 삭제, 수정 (추후 업데이트 가능)

## 사용 기술
- **React Native** (Expo)
- **JavaScript (ES6+)**  
- ~~기타 라이브러리~~ (사용하게 된다면 기입 할 예정띠리따라뽀)

## 프로젝트 구조
\`\`\`
프로젝트 폴더 구조 (지금은 예시입니다)
myTodoApp
├── App.js
├── package.json
└── ...
\`\`\`

## 실행 방법
1. 이 저장소를 클론 받거나 다운로드
2. `npm install` 또는 `yarn install`
3. `npm start` 또는 `yarn start` 후 시뮬레이터 또는 Expo Go 앱에서 실행
